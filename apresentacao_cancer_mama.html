Sobre o Projeto

Desenvolvimento de um modelo de aprendizagem supervisionada para classificação binária de câncer de mama, utilizando dados morfológicos de núcleos celulares.

Tecnologias Utilizadas
Python 3.11.5 TensorFlow Keras Scikit-learn Pandas Kaggle
2/8

Problema de Negócio

Contexto e Relevância

O câncer de mama é uma das principais causas de morte por câncer em mulheres. A detecção precoce é fundamental.

Desafios Atuais
  • Subjetivo: Dependente da experiência
  • Demorado: Análise manual
  • Custoso: Especialistas qualificados
  • Variável: Inconsistências

Estrutura dos Dados

model = Sequential([ Dense(64, activation='relu', input_shape=(30,)), Dropout(0.3), Dense(32, activation='relu'), Dropout(0.3), Dense(1, activation='sigmoid') ])

Justificativas

Feedforward

Dados tabulares sem dependência temporal

2 Camadas Ocultas

Balanceio entre capacidade e simplicidade

Componentes
  • ReLU: Evita vanishing gradient
  • Dropout: Regularização
  • Sigmoid: Output probabilístico
